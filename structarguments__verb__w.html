Detailed Description

Structure used to hold all the info about any parsable launch-arguments which may include sub-verbs and Options.
Uses wchar_t strings for any text-related values.

Warning:
Any reference to np_args_char can be the equivalent of char or wchar_t depending on 2 compiler constants.
See Arguments parser or arguments/macros.h for more info.

Any reference to a sub-verb actually refers to a regular Verb which can be considered as the another verb's children.
