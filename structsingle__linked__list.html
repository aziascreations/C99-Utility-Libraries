Detailed Description

Main list structure that contains pointers to the first, last and current SingleLinkedListNode.
All functions in the sllist module use or return a pointer to this type of structure.

This structure is a 1-to-1 copy of the DoubleLinkedList one.
There are duplicated in order to allow the IDE to raise errors when passing SingleLinkedList instead of DoubleLinkedList or inversely.

Any operations that involves memory allocation/freeing, insertions and data retrieval will use this structure via the LinkedList type definition and a typecast UNLESS the function is specifically meant for double linked lists.

This pointer was implemented in order to mimic PureBasic's lists and to make some generic ordering tasks less tedious. This structure member is also the last since it is planned to be toggleable in the future to help save memory in some edge-cases.
