Detailed Description

Extension of the Linked lists commons module that implements all the required logic for interacting with single linked lists.

It should be noted that the functions whose logic is shared with their counterpart in the Single linked lists module are implemented in the form of a simple macro that links back to the appropriate function in the Linked lists commons module.

Frees the given list and all of its nodes' data using the given callbacks for both the data and nodes.

Parameters
listThe DoubleLinkedList to be freed.
cb_freeDataA function that will free the node's data.
If left as NULL, the data won't be freed !
cb_freeNodeA function that will free the node's structure.
If left as NULL, the standard void free(void *data) function will be used !
Warning
May cause memory leaks if cb_freeData is NULL.
