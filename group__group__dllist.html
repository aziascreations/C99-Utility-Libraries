Detailed Description

Every linked list is composed of these 3 distinct parts.

DoubleLinkedList
Main structure that contains pointers to the first, last and current DoubleLinkedListNode.
All functions in this module use or return a pointer to this type of structure.

DoubleLinkedListNode
Repeatable structure that is pointed to internally by the DoubleLinkedList in order to represent a list entry.
This structure only contains pointer to the data it holds and the next and previous DoubleLinkedListNode.

The data pointer
Also referred to as the node's data, this is a pointer to the data you want the list to hold.
When freeing a list, this is the pointer passed to the freeing function.
