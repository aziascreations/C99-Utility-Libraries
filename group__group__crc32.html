Detailed Description

This module contains small functions to calculate CRC32 hashes.

The currently supported variants are:
● CRC32B    Common variant (7-Zip, ...)

The future variants that are planned:
● CRC32C    Intel intrinsic variant

These functions were implemented to serve as a primitive hashmap hashing algorithm due to their relatively even distribution when cropping the hash of random data.
Some examples for this behaviour are present in the extra/ directory.

Requires either NP_GOODIES_EXPORT_CRC32, NP_GOODIES_EXPORT_HASH or NP_GOODIES_EXPORT_ALL to be defined to be available in an exported dynamic library.
