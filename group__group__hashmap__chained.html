Detailed Description

Chained HashMap implementation that builds off the HashMap Core module.

This type of HashMap has a pre-allocated array of 2 ^ n pointers to potentially chaining ChainedHashMapBucket.
In terms of memory efficiency, this type of internal structure becomes worse than an simple array of ChainedHashMapBucket once you pass the 0.75 load factor mark.

See also
HashMap Core module

DLL_EXP_CHAINED_HASHMAP ChainedHashMap * hashmap_chained_create ( uint8_t  sizePower)

Creates an empty ChainedHashMap with 2 ^ sizePower slots for ChainedHashMapBucket.

Parameters
sizePowerAmount of distinct bucket slots to be available represented as the exponent for a power of two operation.
The size must inclusively be between 2 and 23, which limits the slots count from 4 to 8,388,608 !
Returns
The newly created ChainedHashMap, or NULL if the size was invalid.
