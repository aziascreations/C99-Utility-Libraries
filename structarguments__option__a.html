Detailed Description

Structure used to hold any information about a singular option using char strings for any text-related values.

Warning:
Any reference to np_args_char can be the equivalent of char or wchar_t depending on 2 compiler constants.
See Arguments parser or arguments/macros.h for more info.

◆ registrationCount

int32_t arguments_option_a::registrationCount

Counter used to memorize how many verbs this Option was registered in a Verb in order to help during the memory freeing process.
Once this counter is decremented to 0 after a Verb's options are all released, this Option should be freed from memory !
