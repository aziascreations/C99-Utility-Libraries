Detailed Description

Single header whose main purpose in is to let you easily identify which compiler, OS and ISA is being targeted to potentially implement platform-specific code.

Once included, you can check for various definitions that can help you target specific targets.

For a complete list of scenarios and their associated definitions, check platforms.md.
