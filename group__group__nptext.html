◆ text_copyLine()

char * text_copyLine ( const char *  string,
size_t  stringLength,
char **  nextLine,
size_t *  nextLineMaxLength 
)

Copies the first available line in the given string into a new buffer and returns said buffer as-is.
Depending on the arguments given, it can also easily iterate over every line in a block of text.

Parameters
stringThe string whose first line will be copied and returned.
stringLengthThe given string's length with or without the trailing NULL byte.
May overshoot the actual length if a NULL byte is present in the string.
And may also undershoot the actual length to simulate a truncated string.
nextLineOptional pointer to a string pointer that is set to the start of the next line in the given text. If no more line is present in the given text, it will be set to NULL.
This pointer must never be freed !
nextLineMaxLengthOptional pointer that returns expected remaining string length for nextLine.
Will be ignored if nextLine is NULL.
Returns
The copied line's pointer, or NULL if an error occurred or no line could be found.
Warning
Failure to free the returned string WILL cause memory leaks !
